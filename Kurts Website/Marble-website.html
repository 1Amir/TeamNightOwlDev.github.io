Marble Text Editor

What is Marble?

Marble is an open source text editor with a twist. unlike most text editors, you and a friend can
work on the same project with Marble. Marble is a text editor made with (programminglanguage) and (prog.lang.).

How to download Marble?

Firstly, click the download link. Then, run the installer. Finaly, make an account and login. Done!
